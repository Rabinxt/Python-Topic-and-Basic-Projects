ðŸ”¹ 2. Ways to Import Modules
============================
1. import module
   import math
   print(math.sqrt(9))

2. from module import function
   from math import sqrt
   print(sqrt(9))

3. from module import *
   from math import *
   print(sqrt(9))  # Not recommended in large projects

4. import module as alias
   import math as m
   print(m.sqrt(25))
